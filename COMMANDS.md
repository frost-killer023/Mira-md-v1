# 🎮 Command List - Mira MD Bot

## Utility Commands

### !help
Show help menu with all available commands
```
!help              # Show all commands
!help <command>    # Show specific command help
```

### !ping
Check bot latency and status
```
!ping              # Show latency in ms
```

### !info
Display bot information and features
```
!info              # Show bot details
```

### !menu
Show main menu with categories
```
!menu              # Display menu
```

## AI Commands

### !ai
Ask ChatGPT a question
```
!ai What is Python?           # Ask a question
!ai Explain quantum physics   # Get detailed answer
```

### !imagine
Generate images with AI
```
!imagine a futuristic city    # Generate image
!imagine cat wearing sunglasses
```

## Information Commands

### !weather
Get weather information for a city
```
!weather New York      # Get weather
!weather London        # Temperature, condition, humidity
```

### !translate
Translate text to different languages
```
!translate en Hello           # Translate to English
!translate fr Bonjour        # Translate to French
!translate ar مرحبا          # Translate to Arabic
```

### !wiki
Search Wikipedia
```
!wiki Python programming      # Wikipedia search
!wiki Albert Einstein
```

## Entertainment Commands

### !play
Play music from YouTube
```
!play Bohemian Rhapsody      # Play song
!play Despacito              # Stream audio
```

### !lyrics
Get song lyrics
```
!lyrics Imagine John Lennon   # Get lyrics
!lyrics Shape of You
```

### !meme
Get random meme
```

```

## Admin Commands

### !admin
Open admin panel (owner only)
```
!admin             # Admin dashboard
```

### !ban
Ban user from using the bot
```
!ban 1234567890    # Ban user
```

### !unban
Unban user
```
!unban 1234567890  # Unban user
```

## Command Syntax

- **Prefix**: `!` (configurable)
- **Arguments**: Separated by spaces
- **Cooldown**: Default 3 seconds
- **Permissions**: Some commands are owner-only

## Tips

✅ Use `!help` to see all available commands  
✅ Type `!help <command>` for detailed info  
✅ Commands are case-insensitive  
✅ Some features require API keys in .env  
✅ Contact support for custom commands
